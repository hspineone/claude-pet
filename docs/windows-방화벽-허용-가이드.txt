[이 문서의 목적]
─────────────────────────────────────────────────

ClaudePet 설치는 정상으로 끝났는데, 실행하면 빨간 X 아이콘과 함께
"Failed to launch JVM" 이라는 경고창만 뜨고 캐릭터가 나오지 않는
경우 아래 절차로 1~2분만에 해결할 수 있습니다.


[왜 이런 일이 생기나요?]
─────────────────────────────────────────────────

ClaudePet 은 개인이 만든 작은 프로그램이라 Microsoft 의 공식 서명이
없습니다. 이런 프로그램은 윈도우의 보안 프로그램
(Windows 보안 / Windows Defender, 일상적으로는 "방화벽" 이라고
많이 부릅니다)이 "혹시 위험할 수 있는 프로그램" 이라고 판단해서
실행을 차단할 수 있어요.

이 문서는 ClaudePet 폴더 하나만 예외(허용) 목록에 추가해서
차단을 풀어주는 방법입니다. 전체 보안을 끄는 게 아니라 이 폴더만
허용하는 거라 안전합니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1.  "Windows 보안" 앱 열기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1) 키보드에서 윈도우 로고 키를 한 번 누르세요.
   (키보드 왼쪽 아래 Alt 키 옆, 창문 4개 모양 키)

2) 키를 뗀 상태에서 바로 이렇게 타이핑합니다:

        windows 보안

   자동으로 검색 결과가 뜹니다.

3) 검색 결과에 파란/회색 방패 아이콘의 "Windows 보안" 앱이 나오면
   클릭해서 엽니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2.  "바이러스 및 위협 방지" 로 이동
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1) 창이 열리면 왼쪽에 여러 아이콘이 세로로 나열돼 있습니다.

2) 그 중 방패 모양의 "바이러스 및 위협 방지" 클릭.

3) 오른쪽 페이지에서 조금 아래로 스크롤.
   "바이러스 및 위협 방지 설정" 이라는 큰 제목 아래
   "설정 관리" 라는 파란색 글자(링크)가 보입니다. 클릭.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3.  "제외 항목" 설정 찾기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1) 열린 설정 페이지에서 계속 아래로 스크롤.
   (실시간 보호, 클라우드 제공 보호 등 여러 항목이 지나감)

2) 맨 아래쪽에 "제외 항목" 이라는 제목을 찾습니다.

3) 그 아래의 "제외 추가 또는 제거" 파란 글자 클릭.

4) 노란 방패 + "이 앱이 디바이스를 변경할 수 있도록 허용하시겠어요?"
   라는 관리자 권한 창이 뜨면 "예" 클릭.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4.  ClaudePet 폴더를 예외에 추가 (핵심!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1) 화면 위쪽 "+ 제외 추가" 버튼 클릭.

2) 아래로 작은 목록이 펼쳐집니다:
     • 파일
     • 폴더       ←  이걸 선택하세요
     • 파일 형식
     • 프로세스

3) 폴더 선택 창이 열리면 왼쪽 목록에서 차례로 더블클릭:

     내 PC  →  로컬 디스크 (C:)  →  Program Files  →  ClaudePet

4) ClaudePet 폴더를 한 번 클릭(선택) 한 뒤
   오른쪽 아래 "폴더 선택" 버튼 클릭.

5) 제외 목록에 아래 경로가 표시되면 성공입니다:

        C:\Program Files\ClaudePet


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 5.  ClaudePet 다시 실행
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1) 모든 창을 닫습니다.

2) 윈도우 키 → "claudepet" 입력 → 엔터.
   또는 시작 메뉴에서 ClaudePet 아이콘을 더블클릭.

3) 잠시 후 화면 우측 하단에 작은 병아리 캐릭터가 뜨면 성공!

   작업표시줄 시계 옆 ∧ 화살표 클릭하면 숨겨진 아이콘 영역에
   병아리 모양 트레이 아이콘도 나타납니다. 우클릭하면 숨기기·밥주기·
   종료 메뉴가 뜹니다.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
그래도 "Failed to launch JVM" 이 계속 뜬다면?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

□ PC 를 한 번 재부팅한 뒤 다시 실행해 보세요.

□ 회사/학교 PC 라면 IT 관리자가 별도로 차단했을 수 있습니다.
   관리자에게 "C:\Program Files\ClaudePet\ClaudePet.exe 실행 허용"
   을 요청해 주세요.

□ V3, 알약, 카스퍼스키 등 별도 백신을 쓰고 있다면 그쪽 프로그램에도
   같은 방식으로 예외를 추가해야 합니다. 프로그램마다 메뉴 이름이
   다르지만 대개 "예외 설정", "허용 목록", "차단 해제" 같은 메뉴를
   찾으시면 됩니다.

□ 위 전부 시도했는데도 안 되면, ClaudePet 을 준 사람에게
   "방화벽 예외 추가해도 Failed to launch JVM 이 계속 뜬다" 고
   알려 주세요. 다른 원인 조사가 필요합니다.


끝. 즐거운 시간 되세요 🐣
