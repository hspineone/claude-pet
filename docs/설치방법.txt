[ ClaudePet 설치 방법 ]

※ 업데이트 시 반드시 ClaudePet 앱을 종료한 후 설치해 주세요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[ Windows 설치 방법 ]

1. ZIP 파일 압축 풀기
2. .msi 파일 실행
3. 설치 마법사 진행
   · SmartScreen 경고가 뜨면 "추가 정보" → "실행" 클릭
4. 시작 메뉴 또는 바탕화면 바로가기로 ClaudePet 실행

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[ macOS 설치 방법 ]

1. ZIP 파일 압축 풀기
2. .dmg 파일 열기
3. 앱을 Applications 폴더로 드래그
4. 터미널(Terminal) 에 아래 명령어 입력 (Gatekeeper 해제):
   xattr -cr /Applications/ClaudePet.app
5. Applications 또는 Spotlight 에서 ClaudePet 더블클릭하여 실행
6. 첫 실행 시 접근성 권한 안내가 나오면:
   · "시스템 설정 열기" → 개인정보 보호 및 보안 → 손쉬운 사용
   · ClaudePet 토글 ON
   · 앱 다이얼로그의 "완료 (재시작)" 클릭
